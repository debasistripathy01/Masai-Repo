b tag

Element is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. This was formerly known as the Boldface element, and most browsers still draw the text in boldface

i tag

Element represents a range of text that is set off from the normal text for some reason, such as idiomatic text, technical terms, taxonomical designations, among others

u tag

Element represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation.
